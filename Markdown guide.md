### 목록

- 순서가 없는 목록 : `-` 하이픈 하나
- 들여쓰기 하면 자동으로 다음 목록 생성
  - Tab 해서 다음단으로 들어감
  - Enter쳐서 이어나갈수 있음
- Shift + Tab 을 이용해 돌아올 수 있음
- 목록을 그만 쓸때는 엔터 두번!

1. 순서가 있는 목록
2. enter로 다음 숫자
   1. Tab 누르면 한단 안으로
   2. 내용쓰다가 shift + tab 하면 상단으로 감
3. 끝내려면 Enter 두번
